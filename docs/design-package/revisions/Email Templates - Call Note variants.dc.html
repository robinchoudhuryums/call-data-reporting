How the theming maps
Default (e.g. status check) → navy banner · Verified Shipping / Repeat Resupply → green (#0f8a52) + shipping/resupply detail card · OOP Order → amber (#b7791f) + OOP detail card with total · Close Order → red (#c13030) + reason line. The logo bar, "Update for…" title, and navy Call Details table are identical across all of them — only tplColor/tplSoft/tplDeep and the subform block vary. Repeat Resupply mirrors Verified Shipping (green), so it's not shown separately.
