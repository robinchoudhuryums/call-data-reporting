What changed
  • Bottom sections are now cards — Completion matrix, Active assignments, Quizzes each get a white panel with a titled seclabel + count badge, matching the assign card above.
  • Matrix no longer sprawls — the grid is width:max-content and left-aligned, so a single item's cell + Cov sit right next to the name instead of flying to the right edge.
  • Done cell is a filled rounded chip (not a bare check far away).
  • Quizzes empty-state gets a real placeholder instead of just a lone button.
  • Assign controls use the shared field style (custom chevron, token border) — fixes the native date/select look.
