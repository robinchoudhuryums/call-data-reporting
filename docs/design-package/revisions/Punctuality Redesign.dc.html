What changed
  • Load button is a normal control beside the dates — not a full-width black bar. Added 7d/30d/QTR presets to match Metrics.
  • Stat strip is 4-up (Team on-time · Reps · Avg late · Worst) so it fills the row instead of two cells stranded on the left, and surfaces the lateness magnitude the page hid.
  • Table sits in a card with a titled label, matching every other module.
  • Tri-tone on-time + a per-rep adherence bar make 0% read instantly; Avg/Worst late are color-graded; lunch on-time becomes a chip.
  • Description moves up beside the title instead of stretching across the page top.
Note: the oversized black Load button also appears on Coverage — same fix applies there. Coverage's grid + closed-day handling otherwise looks solid.
