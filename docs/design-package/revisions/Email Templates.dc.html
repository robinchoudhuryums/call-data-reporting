Email-safe rules used
  • Tables for layout — no flex/grid (Outlook drops them).
  • All styles inline — no <style> reliance, no var(); literal hex.
  • System font stack (Arial/Helvetica) — web fonts don't load in most clients.
  • ≤600px content width; single column.
  • Accent rail = 3px top border + left border on resolution; matches the app's inset 3px motif.
  • Badges/pills as inline-block spans (bg + radius) — render everywhere.
  • Buttons as padded <a> (bulletproof-ish); no CSS-only buttons.
